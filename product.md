# Product

## Goal
Система фиксации состояния техники с фото-доказательствами.

## Core Flow
Осмотр → Фиксация → Дефекты → История

## Roles
- Инспектор
- Руководитель

## Inspection Types
- Быстрый
- Полный

## Rules
- Любой "НЕТ" = дефект
- Дефект всегда с фото
- Осмотр нельзя завершить без фото
- Фото только с камеры
- Фиксация гео и времени
