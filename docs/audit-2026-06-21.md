# Audit report - 2026-06-21

## Vehicle detail inspection evidence fixes

- Исправлена выдача `/api/vehicles/:vehicleId/inspections` и общего списка
  осмотров: если у записи есть `completed_at`, API возвращает `completed = 1`,
  чтобы история не показывала завершённый осмотр как “не завершён”.
- Web API client теперь нормализует списки осмотров: `completed`,
  `duration_seconds`, `odometer_value` и `defects_count` приводятся к
  ожидаемым типам перед передачей в карточку техники.
- График “Показания одометра” стал устойчивым к строковым значениям из API и
  строится по всем сохранённым числовым показаниям пробега.
- В истории осмотров длительность отображается в минутах, если есть
  `duration_seconds`, даже когда legacy-флаг `completed` не выставлен.
- Фото дефектов в карточке техники стали кликабельными: миниатюра открывает
  оригинал/WebP/full-size файл в новой вкладке.
