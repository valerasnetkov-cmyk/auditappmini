# Internationalization

## Цель

Интерфейс должен поддерживать пять языков:

```txt
ru -> русский
en -> английский
de -> немецкий
fr -> французский
es -> испанский
```

---

## Принцип

Переводятся системные элементы интерфейса:

- меню;
- кнопки;
- заголовки;
- ошибки;
- подсказки;
- статусы;
- типы осмотров;
- типы фото;
- названия обязательных шагов.

Не переводятся автоматически:

- номер машины;
- название техники;
- название компании;
- имена пользователей;
- комментарии инспектора;
- описание дефектов;
- значение километража.

## Где хранить язык

```txt
companies.default_locale
profiles.locale
```

Приоритет:

```txt
profiles.locale -> companies.default_locale -> browser/device locale -> ru
```

---

## Что не делать

- Не хардкодить тексты в компонентах.
- Не переводить пользовательские комментарии автоматически.
- Не переводить номер машины.
- Не хранить разные языки в разных компонентах без общего словаря.

## Theme translations

Добавить переводы для настроек темы на всех языках `ru`, `en`, `de`, `fr`, `es`.

Ключи:

```json
{
  "settings": {
    "theme": {
      "title": "Тема интерфейса",
      "system": "Как в системе",
      "light": "Светлая",
      "dark": "Тёмная"
    }
  }
}
```

Примеры значений:

```txt
ru: Тема интерфейса / Как в системе / Светлая / Тёмная
en: Interface theme / System / Light / Dark
de: Oberflächendesign / System / Hell / Dunkel
fr: Thème de l’interface / Système / Clair / Sombre
es: Tema de la interfaz / Sistema / Claro / Oscuro
```

Названия пользовательских данных, номера автомобилей, комментарии инспектора и значения одометра не переводятся.

## Тексты единиц измерения пробега

Добавить ключи переводов:

```json
{
  "settings.distanceUnit.title": "Единицы измерения пробега",
  "settings.distanceUnit.km": "Километры",
  "settings.distanceUnit.mi": "Мили",
  "settings.distanceUnit.changedWarning": "Новые осмотры будут использовать выбранную единицу измерения. Старые записи сохранят исходную единицу.",
  "odometer.unit.km": "км",
  "odometer.unit.mi": "mi",
  "odometer.inputLabel": "Введите пробег",
  "odometer.unitMismatchWarning": "Проверьте единицу измерения на фото одометра"
}
```

Пользовательские значения одометра не переводятся, но единица отображается согласно языку интерфейса.

## i18n: vehicle number OCR

Ключи интерфейса для распознавания номера:

```json
{
  "vehicleNumber.recognizeByPhoto": "Распознать номер по фото",
  "vehicleNumber.recognitionFailed": "Номер не распознан",
  "vehicleNumber.confirmRecognized": "Подтвердить распознанный номер",
  "vehicleNumber.retakePhoto": "Сделать фото заново",
  "vehicleNumber.enterLatinOnly": "Введите номер латинскими буквами",
  "vehicleNumber.ocrUnavailable": "Распознавание недоступно. Введите номер вручную."
}
```

Номера автомобилей не переводятся. Финальный номер хранится только латиницей.
